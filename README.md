# lab10
Программирование на C#, лабораторная работа №10 "Текстовый редактор"
Выполнены задания:
10.1	Написан обработчик события нажатия на кнопку «Сохранить». Если текущий текстовый документ не связан с конкретным файлом, то выводится стандартное диалоговое окно сохранения файла, иначе просто сохраняется документ в заданный файл.
10.2	Написан обработчик события нажатия на кнопку создания нового текстового документа. При этом если в текущий текстовый документ были внесены изменения, то выводится диалоговое окно, спрашивающее у пользователя о необходимости сохранения документа.
10.3	Написан обработчик события нажатия на кнопку вызова окна помощи на панели инструментов. При этом выводится диалоговое окно, содержащее информацию о создателе программы.
10.4	Обеспечена возможность задания пользователем с клавиатуры такого значения размера шрифта, которое не указано в раскрывающемся списке.
10.5	Написан обработчик события нажатия на кнопки изменения стиля начертания на курсив и подчеркнутый. Достоверно то, что при перемещении по тексту кнопки отображения начертания символов правильно отображают стиль начертания текущего символа.
10.6	Добавлены на панель инструментов кнопки изменения цвета текста и цвета выделенного текста (заливки) и обеспечены их функциональность. Для выбора цвета используется элемент управления ColorDialog.
10.7	Добавлена на панель инструментов кнопка вызова стандартного диалогового окна изменения шрифта, обеспечена ее функциональность. Использован элемент управления FontDialog.
10.8	Обеспечена функциональность оставшихся кнопок выравнивания абзаца. Достоверно то, что при перемещении по тексту кнопки выравнивания правильно отображают способ выравнивания текущего абзаца.
10.9	Созданы и отображены иконки для кнопок изменения цвета текста и цвета выделенного текста (заливки).
10.10	Обеспечена функциональность кнопок редактирования текста: вырезать, копировать и вставить.
10.11	Разработано главное меню, реализующее все функциональные возможности приложения, обеспечена его работоспособность.
